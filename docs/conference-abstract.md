# Conference Abstract

**Benchmarking Reference-Guided Cell Type Annotation Across Neuropsychiatric Single-Cell Datasets**

Single-cell transcriptomics studies employ diverse quality control (QC) and cell type annotation (CTA) methodologies, creating challenges for reproducibility and cross-study meta-analysis. We surveyed QC and CTA approaches across 147 human and 51 mouse brain datasets, revealing substantial heterogeneity in filtering thresholds, doublet detection algorithms, and marker gene selection. Aligning author-provided annotations across studies is further complicated by inconsistent nomenclature and varying granularity. To address these inconsistencies, we developed a standardized pipeline implementing data-driven QC and reference-guided hierarchical cell type annotation, now deployed for over 200 experiments in the Gemma gene expression database. We benchmarked three classifiers (scVI with random forest, scVI with k-nearest neighbors, and Seurat's Gaussian kernel label transfer) across 16 human and 7 mouse query datasets spanning multiple brain regions and disease contexts. Confidence thresholding exerted the strongest influence on annotation performance: scVI RF was highly sensitive to increasing cutoffs (macro F1 dropping from ~0.96 to ~0.38 in mouse), while Seurat and scVI kNN both maintained near-baseline performance up to cutoff 0.75. scVI-based classifiers offer a computational advantage by projecting query datasets into a pre-trained embedding space, running approximately 2x faster and using 2x less memory than Seurat. Performance varied by cell type resolution, with coarser groupings achieving 15-20% higher accuracy than subclass predictions. This standardized framework provides a foundation for consistent reference-guided annotation across diverse neuropsychiatric datasets and highlights key practical challenges for cross-study comparability.
